- added SQL setup file: work_order_photos_setup.sql

How it works:

Deploy:
1. Upload this build
2. Run work_order_photos_setup.sql in Supabase SQL editor
3. Deploy:
   cd ~/Downloads/zummee_maintenance_workflow_v67_photo_uploads
   supabase functions deploy upload-work-order-photo --project-ref slcwuuwyrgnmlmxpcaim --no-verify-jwt

Optional:
   supabase secrets set WORK_ORDER_PHOTO_BUCKET=work-order-photos
