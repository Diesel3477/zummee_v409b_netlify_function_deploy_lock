Zummee v408 - Board item photo attachments

Base: v407_board_item_create_work_order

What changed:
- Board Member Hub New Item form supports attaching one image while creating a board item.
- Existing submitted board records can be opened and updated with additional photos.
- Photos upload to Supabase Storage bucket: board-item-photos.
- Photo metadata is stored on notification_events.metadata.photos so the board notification remains the authority record.
- Board item list displays a photo badge when records have attachments.

SQL:
- Run ZUMMEE_V408_BOARD_ITEM_PHOTO_STORAGE.sql once if the board-item-photos bucket/policies do not already exist.
