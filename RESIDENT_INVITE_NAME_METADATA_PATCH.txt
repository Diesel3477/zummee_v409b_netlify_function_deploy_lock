Resident invite function patch

What changed:
- New resident invites now send Resident name metadata into Supabase Auth.
- Existing resident accounts are updated with name/display metadata and linked back to resident_directory.
- Redirect URL must come from the RESIDENT_REDIRECT_URL secret.

Important:
Redeploy this function after copying it into your Supabase project:

supabase functions deploy send-resident-invites --project-ref slcwuuwyrgnmlmxpcaim --no-verify-jwt
