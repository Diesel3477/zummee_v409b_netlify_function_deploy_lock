v59 contact options upgrade

What changed:
- vendor email now uses a single "Open Contact Options" link
- new public page added: vendor_contact.html
- vendor_contact.html provides:
  - Call Manager
  - Text Manager
  - Mark Job Complete

Why:
Mobile email clients often block or ignore tel:/sms: actions inside the email itself.
A normal web page is much more reliable for one-tap calling and texting.

Deployment:
1. Upload this build
2. Redeploy assign-vendor-email:
   cd ~/Downloads/zummee_maintenance_workflow_v59_contact_options
   supabase functions deploy assign-vendor-email --no-verify-jwt
3. Optional if your public page URL differs:
   supabase secrets set VENDOR_CONTACT_BASE_URL=https://zummee.net/vendor_contact.html
